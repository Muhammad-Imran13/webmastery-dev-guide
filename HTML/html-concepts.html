1. Document Architecture & Metadata

A standard HTML5 document follows a strict hierarchy. The <head> contains machine-readable information (metadata) like SEO titles, character sets, and viewport settings for mobile responsiveness, while the <body> contains the visible content.

HTML Technical Code

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Web Tutorial">
    <title>Professional Page</title>
</head>
<body>
    <header>Welcome to HTML5</header>
</body>
</html>

2. Typography & Text Semantics

HTML provides semantic tags to define the importance of text. Headers <h1> through <h6> define the content's outline, while formatting tags like <strong> and <mark> provide visual and structural emphasis.

<h1>Primary Heading</h1>
<h3>Sub-Section Heading</h3>
<p>Regular text with <strong>Bold Emphasis</strong>.</p>
<p>Using <mark>Highlighting</mark> for attention.</p>
<p>Chemical Formula: H<sub>2</sub>O</p>
<p>Math Equation: X<sup>2</sup> + Y<sup>2</sup></p>

4. Hypermedia & Anchors

The <a> tag connects the web together. Attributes like target="_blank" control link behavior, while the <img> tag integrates visual assets with mandatory alt text for accessibility.

<!-- External Navigation -->
<a href="https://github.com" target="_blank">GitHub Profile</a>

<!-- Image with optimization -->
<figure>
  <img src="images/workplace.jpg" alt="Tech Banner">
  <figcaption>Fig 1: Modern Workplace</figcaption>
</figure>

5. Advanced Data Tables

Tables organize complex data into rows (<tr>), headers (<th>), and cells (<td>). Semantic table structures include <thead> and <tbody> for better accessibility.

<table>
  <thead>
    <tr>
      <th>Skill</th>
      <th>Level</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>HTML5</td>
      <td>Expert</td>
    </tr>
    <tr>
      <td>CSS3</td>
      <td>Advanced</td>
    </tr>
  </tbody>
</table>

6. User Forms & Input Validation

Forms are the primary way to collect user data. Modern HTML5 includes built-in validation attributes like required, placeholder, and specific input types like email and password.

<form>
  <label>Email Address:</label>
  <input type="email" placeholder="user@example.com" required>
  
  <label>Experience:</label>
  <select>
    <option>Beginner</option>
    <option>Professional</option>
  </select>

  <button type="submit">Register</button>
</form>



7. Semantic Layout Elements

Semantic HTML5 elements like <main>, <section>, and <article> describe their purpose to the browser and search engines, replacing generic <div> tags.

<nav>Site Navigation</nav>
<main>
  <section>
    <article>Blog Content</article>
  </section>
  <aside>Related Links</aside>
</main>

8. Character Entities & Symbols

Reserved characters in HTML (like angle brackets) must be replaced with entities. They are also used to display symbols and emojis that are not present on a standard keyboard.

<p>Equation: A &lt; B &amp; C &gt; D</p>
<p>Copyright: &copy; 2025 WebDev</p>
<p>Currency: &euro; &pound; &yen;</p>
<p>Emoji: &#128640; (Rocket)</p>
